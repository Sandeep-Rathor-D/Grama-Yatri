# 🚌 Grama Yatri  
### Community‑Powered Rural Bus Tracker (Android Application)

---

## 📌 Project Overview

Grama Yatri is an Android application designed to improve public transportation access in rural areas.  
The app enables users to view bus schedules, track routes, report delays, and share real-time travel updates within their community.

This project aims to enhance rural mobility through technology-driven, community-powered transport tracking.

---

## 🎯 Problem Statement

In many rural areas:

- Bus timings are uncertain
- No real-time tracking is available
- Passengers lack delay information
- Communication gaps cause inconvenience

Grama Yatri solves this by providing a centralized platform for rural bus tracking and updates.

## 🔧 Installation & Setup Guide

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sandeep-Rathor-D/Grama-Yatri.git
```

### 2️⃣ Open in Android Studio

- Open Android Studio  
- Click **Open**  
- Select the cloned folder  
- Wait for Gradle sync  

### 3️⃣ Run the Application

- Connect Android device or start Emulator  
- Click ▶ Run button  

---

## 📦 APK Installation

If using the Release APK:

1. Transfer `Grama-Yatri-Release.apk` to Android device  
2. Enable **Install from Unknown Sources**  
3. Tap the APK file  
4. Install and launch
